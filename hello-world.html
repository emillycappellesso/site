Olá Mundo em Diferentes Linguagens

Explore como escrever "Olá Mundo" em várias linguagens de programação. Cada exemplo inclui uma breve descrição e nível de dificuldade.

Python Fácil

Python é uma linguagem de alto nível, interpretada e de propósito geral. É conhecida por sua sintaxe simples e legível.

print("Olá, Mundo!")
JavaScript Fácil

JavaScript é a linguagem principal da web, permitindo interatividade em páginas web e aplicações.

console.log("Olá, Mundo!");
Java Médio

Java é uma linguagem orientada a objetos, compilada e multiplataforma, muito usada em aplicações empresariais.

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Olá, Mundo!");
    }
}
C# Médio

C# é uma linguagem moderna da Microsoft, usada principalmente para desenvolvimento Windows e jogos com Unity.

using System;

class Program {
    static void Main() {
        Console.WriteLine("Olá, Mundo!");
    }
}
HTML Fácil

HTML é a linguagem de marcação padrão para criar páginas web e aplicações web.

<!DOCTYPE html>
<html>
<head>
    <title>Olá Mundo</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
</body>
</html>
PHP Médio

PHP é uma linguagem de script do lado do servidor, amplamente usada para desenvolvimento web.

<?php
echo "Olá, Mundo!";
?>
Ruby Fácil

Ruby é uma linguagem dinâmica e orientada a objetos, conhecida por sua elegância e simplicidade.

puts "Olá, Mundo!"
Go Médio

Go é uma linguagem compilada desenvolvida pelo Google, conhecida por sua simplicidade e eficiência.

package main

import "fmt"

func main() {
    fmt.Println("Olá, Mundo!")
}
Rust Difícil

Rust é uma linguagem de sistemas que enfatiza segurança, concorrência e performance.

fn main() {
    println!("Olá, Mundo!");
}
Swift Médio

Swift é a linguagem moderna da Apple para desenvolvimento iOS, macOS e watchOS.

print("Olá, Mundo!")
